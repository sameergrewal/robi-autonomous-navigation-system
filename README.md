# ROBI Autonomous Navigation System

An autonomous Raspberry Pi robot capable of obstacle detection,
ledge detection, and autonomous navigation using custom C drivers
and sensor monitoring.

## Technologies
- C
- Python
- Raspberry Pi
- I2C Communication
- GPIO
- POSIX Threads (pthreads)

## Features
- Motor control
- Wall detection
- Ledge detection
- Autonomous navigation
- Multithreaded sensor monitoring

## Repository Structure

- C_Files: Low-level drivers and robot APIs
- Python_Files: Python control scripts
- Tests: Testing and validation code
- Documentation: Project write-up
